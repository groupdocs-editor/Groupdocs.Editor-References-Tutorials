# 인터랙티브 폼 Java 만들기 – GroupDocs.Editor를 사용한 폼 필드 편집

이 허브에서는 GroupDocs.Editor를 사용하여 **Java 기반 인터랙티브 폼** 솔루션을 만드는 데 필요한 모든 것을 확인할 수 있습니다. 문서 중심 웹 앱을 구축하든, 자동화된 폼 처리 파이프라인을 만들든, 혹은 프로그래밍 방식으로 폼 필드를 조작해야 하든, 이 튜토리얼은 실제 시나리오를 단계별로 안내합니다. 폼 필드 데이터를 편집하고, 수정하며, 보존하는 방법을 배우면서 사용자 경험을 원활하고 신뢰성 있게 유지할 수 있습니다.

## 인터랙티브 폼 Java 만들기 – 개요

GroupDocs.Editor for Java는 개발자에게 문서를 로드하고, 레거시 및 최신 폼 필드를 다루며, 인터랙티브성을 잃지 않고 결과를 저장할 수 있는 강력한 API를 제공합니다. 아래 가이드를 따라 하면 다음을 수행할 수 있습니다:

* 인터랙티브 폼 요소가 포함된 Word 또는 PDF 파일을 로드합니다.  
* 잘못되었거나 손상된 폼 필드 컬렉션을 감지하고 복구합니다.  
* 프로그래밍 방식으로 필드 값을 설정, 읽기 또는 삭제합니다.  
* 폼 내용을 업데이트하면서 원본 레이아웃과 스타일을 보존합니다.
